4. Unsupervised Learning Task (“Wholesale Customers”)

Identify homogeneous population groups and dimension reduction techniques for “wholesale customer” dataset. We decided to exclude “Channel” and “Region”, as categorical variabels may not work well on distance based algorithms.

4a. K-Means Clustering


Conduct K-Means to divide dataset into K clusters with the nearest means. First, we need to identify the number of K. We will use “The Elbow Method”.

# Remove 'Channel' and 'Region' variables from the dataset

wholesale_data_without_cat <- wholesale_data[, !(names(wholesale_data) %in% c("Channel", "Region"))]

 

# Compute the within-cluster sum of squares (WCSS) for different values of K

wcss <- vector()

for (i in 1:10) {

  kmeans_model <- kmeans(wholesale_data_without_cat, centers = i, nstart = 10)

  wcss[i] <- kmeans_model$tot.withinss

}

 

# Plot the elbow method graph

plot(1:10, wcss, type = 'b', pch = 19, frame = FALSE, xlab = 'Number of Clusters (K)', ylab = 'Within-cluster Sum of Squares (WCSS)', main = 'Elbow Method')

The elbow point is K=5, where the slope gets flatter.


# Fit K-means model with k=5

kmeans_model <- kmeans(wholesale_data_without_cat, centers = 5, nstart = 10)

 

# Get cluster assignments for each observation
